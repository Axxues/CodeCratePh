Privacy Policy

1. Data Handling

We only collect data required for transactions (Email, Name, Payment Reference). We do not store source code on our own servers; all downloads are via secure, ephemeral links.

2. Seller Anonymity

Sellers may choose to display a pseudonym. However, real identities must be verified with CodeCrate PH admin to receive payouts (KYC).

Terms of Service

1. Educational Use Only

CodeCrate PH sells "Study Templates". We do not condone submitting purchased code as your own original work (plagiarism). These are intended as references to help you build your own system.

2. Digital Goods Policy

Due to the nature of digital source code, all sales are final. Refunds are only issued if the code is proven to be corrupt or significantly different from the description.

3. Seller Commission

CodeCrate PH takes a flat 20% commission on every completed sale to cover server costs and payment gateway fees.
